A simple blog powered by topolr

## How to run

### step 1

`$ npm install topolr-server -g`

### step 2

`$ git clone https://github.com/topolr/topolr-blog`

### step 3

`$ npm install topolr-builter -g`

### step 4

```
$ cd topolr-blog
$ topolr-server add blog [path]/topolr-blog
$ topolr-builter develop
$ topolr-server run

```
